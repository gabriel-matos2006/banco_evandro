#  Sistema de Caixa Eletrônico em Python

##  Descrição
Sistema completo de caixa eletrônico (ATM) desenvolvido em Python utilizando conceitos avançados de Programação Orientada a Objetos.

##  Conceitos POO Implementados
-  **Pacotes**: Organização modular do código
-  **Classes e Objetos**: Estrutura completa com instanciação
-  **Herança**: ContaCorrente e ContaPoupanca herdam de Conta
-  **Polimorfismo**: Método `sacar()` com comportamentos diferentes
-  **Agregação**: Banco agrega múltiplas contas e clientes
-  **Composição**: Conta possui Histórico (não existe sem a conta)
-  **Encapsulamento**: Atributos privados com getters/setters

##  Funcionalidades
1. Criar Cliente
2. Criar Conta (Corrente/Poupança)
3. Depositar dinheiro
4. Sacar dinheiro (com validações)
5. Consultar saldo
6. Ver histórico de operações
7. Listar todas as contas
